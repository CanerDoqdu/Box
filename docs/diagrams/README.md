# Interaktif Mimari Diyagram

Acilis dosyasi:

- [box-architecture-interactive.html](box-architecture-interactive.html)

## Neden bu format?

Bu dosya Mermaid degil, dogrudan interaktif HTML oldugu icin:

- Fare ile surukleyebilirsin
- Mouse tekerlegi ile zoom yapabilirsin
- 3 farkli gorunum arasinda tek tikla gecis yapabilirsin

## Gorunumler

- 1) Genel Mimari: tum sistemi tek ekranda sade bloklarla gosterir
- 2) Liderlik Akisi: Jesus -> Athena -> Queue -> Worker zincirini sirali gosterir
- 3) Mod Gecisi: Strategic, Tactical, Recovery arasindaki gecisleri aciklar

## Kullanım

1. VS Code icinde dosyayi ac.
2. Sag tik -> Open in Browser (veya varsayilan tarayicida ac).
3. Sol tik basili tutup surukle.
4. Mouse tekerlegi ile zoom yap.
5. Ustteki butonlardan diyagram tipini degistir.
